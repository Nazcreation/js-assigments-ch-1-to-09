    Explanation:

    1- Post-increment (num++): Returns the current value, then increments by 1.
    2- Pre-increment (++num): Increments by 1, then returns the new value.
    3- Post-decrement (num--): Returns the current value, then decrements by 1.
    4- Pre-decrement (--num): Decrements by 1, then returns the new value.
